# Masib Collection

A premium, artisanal luxury footwear brand based in Kano, Nigeria. This project is a multi-page responsive website showcasing handcrafted shoes, artisanal craftsmanship, and the heritage of Northern Nigerian leatherworking.

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **Interactive Collection**: Filter and sort products by category and price.
- **Product Modals**: Detailed views for each shoe including care instructions and materials.
- **SEO Optimized**: Advanced meta tags, Open Graph support, and structured sitemap.
- **PWA Ready**: Web manifest included for mobile app-like experience.
- **Smooth Animations**: Intersection Observer API used for scroll-reveal effects.

## Project Structure

- `index.html`: Main landing page (Splash).
- `home.html`: Primary storefront overview.
- `products.html`: Full shop with filtering and search logic.
- `about.html`: Brand story and founder profile.
- `craftmanship.html`: Detailed overview of the shoemaking process.
- `contact.html`: Inquiry form and atelier locations.
- `style.css`: Custom luxury design system using CSS variables.
- `main.js`: Navigation, filtering, and modal logic.

## Development

To run this locally, simply open `index.html` in any modern web browser.
